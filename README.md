# GreenCity QA Test Cases 

## Короткий опис завдання
Цей репозиторій створено в рамках практичного завдання з тестування ПЗ. Він містить тест-кейси для перевірки функціональності сторінки подій проєкту GreenCity. 

## Об'єкт тестування
* **Проєкт:** GreenCity
* **Сторінка, що тестується:** [GreenCity Events Page](https://www.greencity.cx.ua/#/greenCity/events)

## Структура репозиторію
У папці `test-cases` знаходиться файл формату `.md` з розписаними тест-кейсами, що включають передумови, кроки, тестові дані та очікуваний результат.
